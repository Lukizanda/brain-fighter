# Brain Fighter

Educational shooter built on a Roblox TPS template.

## Getting Started

To build the place from scratch, use:

```bash
rojo build -o "BrainFighter.rbxlx"
```

Next, open `BrainFighter.rbxlx` in Roblox Studio and start the Rojo server:

```bash
rojo serve
```

For more help, check out [the Rojo documentation](https://rojo.space/docs).

See [TEMPLATE.md](TEMPLATE.md) for what the underlying TPS template includes.
